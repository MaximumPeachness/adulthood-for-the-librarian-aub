# 📸 Displaying Images from GitHub (How to Use)

To display an image stored in your GitHub repository inside an HTML file, you must use the **raw file URL**. GitHub cannot serve normal repo links as direct images, so the `raw.githubusercontent.com` format is required.

***

## ✅ **Steps**

1.  Go to your image file in GitHub
2.  Click **“Download raw file”** or open it and copy the **raw** URL
3.  Paste that URL into the `src` of your `<img>` tag

***

## 📄 **Example HTML**



Replace the placeholders:

*   `USERNAME` → your GitHub username
*   `REPOSITORY` → the repo name
*   `BRANCH` → usually `main`
*   `PATH/FILE.jpg` → folder and file name

***

## 💡 Tips

*   Works with `.jpg`, `.png`, `.gif`, `.svg`
*   If GitHub caches an older version, add a version tag:
    ?v=1
*   Make sure your repo is **public**, or the image won’t load
